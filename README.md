簡介：
    本組為此電商系統資料庫建置了10個Table，分別為
    
    1. Customer：儲存顧客資料。
    
    2. Product：儲存產品資料。
    
    3. Shipment：儲存物流資料。
    
    4. Store：儲存商店資料。
    
    5. Manufacter：儲存製造商資料。
    
    6. Restok_Order：儲存補貨訂單資料。
    
    7. Restok_Detail：儲存補貨訂單明細資料。
    
    8. Customer_Order：儲存顧客訂單資料。
    
    9. Order_Details：儲存顧客訂單明細資料。
    
    10. Inventory：儲存商品庫存量資料。
